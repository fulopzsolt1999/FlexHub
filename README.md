Steps to run the project:

* Inside backend folder:
  1. composer install
  2. update .env file -> database setup
  3. php artisan migrate
  4. php artisan db:seed
  5. php artisan serve
* Inside frontend folder:
  1. npm install
  2. npm run dev

---

Used components versions:

* Vue CLI v5.0.8
* Laravel Framework v12.3.0
* PHP v8.4.0
* TypeScript v5.8.0
